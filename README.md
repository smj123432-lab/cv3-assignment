# 라방바 데이터랩 - 채용 과제

## 실행 방법

```bash
npm install
npm run dev
```

이후 터미널에 표시되는 주소(기본 http://localhost:5173)로 접속합니다.

## 구현 범위

- 유형 토글(라방 / 홈쇼핑)로 목록 전환
- 유형별 최대 10개 방송을 테이블로 표시
- 컬럼: 순위, 방송정보(제목+채널), 분류, 방송시간, 조회수(라방)/시청률(홈쇼핑), 판매량, 매출액, 상품수
- 로그인 필요 항목(조회수/판매량/매출액)은 🔒 표시로 대체 (자물쇠 인증 로직은 미구현, 과제 안내에 따름)
